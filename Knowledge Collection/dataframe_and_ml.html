When adding the constant 10 to all of our features, we basicially only shift the regression line. However, the slope of the regression remains similar (coefficents) because the features remain at the same relative location to each other. The intercept is the predicted value y in case all the x predictors are equal to 0. When we run our regression model, we want to minimize the error (difference between predicted y and actual y) - as the results indicate we manage to find the appriate slope (b1, b2, ..) that minimizes the error. However, the intercept is still very different because we have shifted the regression line to the right. RMSE remains the same because we only shift all the features, but the regression line will look exactly the same - only in another "cloud of data points", this cloud is shifted but the difference between our prediction and the actual value y is still the same.
